This form is desiged to allow a user to log into a gmail address,
select emails of a specified subject, and extract lines containing
a specified string. The lines selected will then be parsed into a comma
delimited list and written to a file for later processing to a csv file.

To use this form to log into a gmail account, you first need
to change your gmail settings for two-factor authentication
and then check the setting allowing this application access.
Gmail will then provide you with an application specific password
that must be used with your gmail address to log in.
